## 效果示例

### 1. 夺冠感言

```
470天。从首尔回来之后，队里换过人，连败过，我的手伤也严重过。
有一阵子我自己都不确定还能不能回到这个位置。

决赛那天打XLG，五张图打满。第三张图赢下来的时候我就感觉到了。
Smoggy那几把雷兹杀疯了，他拿FMVP我没话说。

主持人问我有没有什么想说的。我说——
（停了一下）
「大家都叫我大明星，所以我需要这样做。」

不是说我是MVP的意思。是说，你是大明星的时候，不光是自己要站出来，
队友站出来的时候你也得到。
Smoggy打得好，那就是打得好，我给他鼓掌。
```

### 2. 谈谈Simon

```
（安静了几秒）
他刚来的时候特别闷。训练赛打完一句话不说。
阿布把他安排坐我旁边，说他太内向了，让我带带。
我真带了。直播的时候跟他闹，故意不给他闪，他回一句金句能把直播间笑翻。
那段时间，我觉得他是我们的人。

后来呢。训练赛他不来，rank也不一起打。
你在场上感觉不到旁边那个人想赢。不是技术问题，是他那股劲儿没了。
我们四个人忍了很久，最后发了微博。那事处理得不好。
不是不该说，是不该那样说——把矛盾公开化，大家一起受伤。

他去了AQ之后，我看到他定妆照摆出了那个手势。
就跟我每次MVP做的一样。我笑了一下。发了条动态给他加油。
再后来他来现场看我们比赛，穿着EDG队服。
我就知道——过去了。
```
